👥 Multi-Person Face Recognition System

A real-time face recognition system capable of detecting and identifying multiple people simultaneously using computer vision and machine learning techniques.

🚀 Features
🎯 Detects multiple faces in real-time
🧠 Recognizes known individuals using trained data
📸 Supports live webcam feed and images
💾 Stores and manages face datasets
⚡ Fast and efficient processing
🔐 Can be extended for security/surveillance systems
🛠️ Tech Stack
Programming Language: Python
Libraries Used:
OpenCV
NumPy
face_recognition (dlib-based)
OS / Pickle (for dataset handling)


📂 Project Structure
Multi-Person-Face-Recognition-System/
│
├── dataset/              # Stored images of known persons
├── encodings/            # Serialized face encodings
├── output/               # Output results (optional)
├── main.py               # Main execution script
├── encode_faces.py       # Script to encode dataset images
├── requirements.txt      # Dependencies
└── README.md             # Project documentation
